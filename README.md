# League of Legends Champion Viewer

A Node.js + Express web application that displays all playable League of
Legends champions using the Riot Games API.

This app showcases my skills in building a dynamic web application using
Node.js and Express, plus integration with the Riot Games API.

## Features

-   Fetches real-time champion data

## Getting Started

### Prerequisites

-   Node.js v14+
-   NPM or Yarn
-   Riot API Key

### Installation

    git clone https://github.com/RenzoCardoza/LeagueofLegends_App
    cd LeagueofLegends_App
    npm install

### Environment

Create a `.env` file:

    RIOT_API_KEY=your_api_key_here

### Run

    npm start

## Usage

Visit `http://localhost:3000` and browse champions.
